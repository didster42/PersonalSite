First post

One of the things I put on my to-do list for 2021 was writing more. I had been thinking about doing something like this for quite some time for a few reasons

  1. I felt that it would be cool to write about my experiences and document them to look back at them in the future.
  2. I thought it would allow me to learn to articulate things better and to organize my thoughts.
  3. I googled reasons to start a blog because I couldn't think of anything to write, which adds to why I should start writing more.

Now that I have reserved a corner for myself on the internet, the next challenge is actually to write something.
